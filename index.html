Sortierverfahren

Beginn 1. Bubble Sort Bei diesem Sortierverfahren werden die Elemente von links nach rechts gelesen und mit dem rechten Element verglichen. Wenn das rechte Element kleiner ist, werden die Elemente vertauscht. Danach wird es mit dem nächsten Element verglichen, bis es am Ende der Liste angelangt ist. Am Ende der Phase ist die Liste ansteigend oder absteigend sortiert. Diese Phase wird so lange wiederholt bis die Liste sortiert ist. Der Name des Verfahrens ist Bubble kommt davon, weil die Elemente wie Blasen nach oben verschoben werden, also an das Ende der Liste gebracht werden. 2. Radix Sort Bei diesem Sortierverfahren wird die Liste in verschiedene Gruppen eingeteilt. Jede Gruppe besteht aus Elementen mit dem gleichen Wert von der Zahl. Die Elemente in der Gruppe werden sortiert. Die Elemente werden dann wieder zusammengefasst. Dadurch wird die Liste sortiert. Das Sortieren folgt dabei nach Grössen der Zahlen von links nach rechts. Das heisst die Gruppe mit dem kleinsten Wert wird zuerst sortiert. Nach der ersten Phase werden die Elemente dann der Grösse nach aus ihren Gruppen genommen. In dieser Reihenfolge geht es in die nächste Phase. So wird jede Phase durchlaufen, bis schlussendlich die ganze Liste sortiert ist. 3.Selection Sort Das ist das Sortieren durch direktes Auswählen. Man suchtdabei entweder nach dem grössten oder nach dem kleinsten Element. Wenn man nach dem kleinsten Element sucht, geht man durch die Liste. Wenn man das kleinste Element gefunden hat vertauscht man es mit dem ersten Element der Liste. Bei den nächsten Durchläufen muss man dieses Element nicht mehr involvieren. Für die nächste Phase nimmt man das Element rechts von dem "fertigsortierten" Element und fährt so weiter und sucht immer das kleinste Element bis die ganze Liste sortiert ist.
